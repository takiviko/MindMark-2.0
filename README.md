# MindMark 2.0

A Kotlin reboot of my MindMark app I created using Java.

Technologies used:
- ConstraintLayout
- DataBinding
- Fragments
- SafeArgs Plugin
- RoomDatabase
- Coroutines for database operations
- RecyclerView
- RecyclerView
- RecyclerView ClickListener
- RecyclerView Header
